# Iris Flower Classification 🌸

This project is a Machine Learning model that classifies iris flowers into different species based on their features such as sepal length, sepal width, petal length, and petal width.

## 📌 Project Overview

The Iris Flower Classification project uses the famous Iris dataset to train a classification model. The model predicts the species of an iris flower:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📂 Dataset

The dataset contains flower measurements including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species
